# Quickly start
```
git clone https://github.com/YLX621/Counter.git
```

And

```
cd backend
```



```
go run server.go
```

```
cd ..
```

```
cd frontend
```

```
cd calculator
```

```
npm install
```

```
npm run serve
```
Listen your port :8080
